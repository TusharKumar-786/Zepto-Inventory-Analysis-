# Zepto Inventory Analysis using PostgreSQL

## Project Overview

This project analyzes Zepto inventory data using PostgreSQL.

The objective was to explore product inventory, pricing, discounts, and stock availability and generate business insights using SQL.

---

## Dataset Overview

- Total Rows: 3,732+
- Database: PostgreSQL
- Dataset: Zepto Product Inventory

Columns included:

- Category
- Product Name
- MRP
- Discount %
- Available Quantity
- Discounted Selling Price
- Weight
- Out of Stock
- Quantity
